 Dynamic Pricing System

This project is a web-based Dynamic Pricing System developed as a minor project for MCA.

Features
- Admin can add products and manage stock
- Prices update dynamically based on stock levels
- Rule-based pricing logic
- Backend using Node.js and Express
- MongoDB used for data storage
- Simple frontend using HTML, CSS, and JavaScript

Pricing Rules
- Stock < 10 → Price increases by 10%
- Stock > 50 → Price decreases by 5%
- Otherwise → Price remains unchanged

Technologies Used
- HTML, CSS, JavaScript
- Node.js, Express
- MongoDB Atlas
